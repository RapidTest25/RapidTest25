### Untuk Pemula:

1. **YOLO Badge** (Paling Mudah)
   - Buat repository baru
   - Buat branch baru
   - Push perubahan ke branch tersebut
   - Buat PR dari branch ke main
   - Merge tanpa review

2. **Heart On Your Sleeve** (Mudah)
   - Cari repository populer
   - Beri reaction â¤ï¸ di berbagai issues/PRs/discussions
   - Ulangi hingga dapat badge

3. **Quickdraw** (Mudah)
   - Di repo Anda, buat issue sederhana
   - Dalam 5 menit, buat fix dan close issue tersebut
   - Atau buat PR dan merge dalam 5 menit

4. **Pair Extraordinaire** (Sedang)
   - Ajak teman kolaborasi
   - Buat commit dengan co-author
   - Format commit message:
     ```
     Your commit message
     
     Co-authored-by: Friend Name <friend@email.com>
     ```

5. **Pull Shark** (Berkelanjutan)
   - Mulai kontribusi ke open source
   - Cari "good first issue" di repositories populer
   - Buat PR berkualitas
